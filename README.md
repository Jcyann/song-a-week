# 🎵 Song-a-Week Challenge

A gamified web app where musicians commit to learning one song per week, track their progress, earn badges, and share with a supportive community.

## 🌐 Live Demo

[Deploy to Vercel](#) — Or run locally with any static server.

## ✨ Features

### For Musicians
- **Weekly Song Commitment** — Pick a song, set a target date, track your journey
- **Progress Tracker** — Mark lessons completed, practice sessions logged
- **Badge System** — Earn badges for streaks, completions, and community engagement
- **Share Your Progress** — Record/upload covers, get feedback from community
- **Weekly Challenges** — Participate in themed challenges (#FolkWeek, #BluesWeek)

### Community Features
- **Song Suggestions** — Browse what others are learning
- **Duet/Collab Requests** — Find musicians to collaborate with
- **Leaderboard** — Weekly and all-time most active learners
- **Discussion Forums** — Tips, tablature sharing, encouragement
